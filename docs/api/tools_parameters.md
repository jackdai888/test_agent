# 工具参数说明文档

## 📋 工具分类说明

### 1. 设备连接工具

#### connect_devices
- **功能**: 连接Android设备并获取设备信息
- **参数**: 无
- **返回**: 设备连接状态和设备信息
- **示例**: "连接设备并告诉我设备信息"

### 2. 基础操作工具

#### input_text
- **功能**: 在文本框输入文字
- **参数**: 
  - `text` (字符串): 要输入的文字内容
- **示例**: "在搜索框输入'微信'"

#### swipe_screen
- **功能**: 滑动屏幕
- **参数**:
  - `start_x` (数字): 起始点X坐标
  - `start_y` (数字): 起始点Y坐标  
  - `end_x` (数字): 终点X坐标
  - `end_y` (数字): 终点Y坐标
  - `duration` (数字, 可选): 滑动时间(毫秒)，默认500
- **示例**: "从屏幕中间向上滑动"

#### press_key
- **功能**: 按下设备按键
- **参数**:
  - `key_name` (字符串): 按键名称
    - 常用按键: "back"(返回), "home"(主页), "menu"(菜单)
    - 音量键: "volume_up"(音量+), "volume_down"(音量-)
    - 电源键: "power"(电源)
    - 方向键: "dpad_up"(上), "dpad_down"(下), "dpad_left"(左), "dpad_right"(右)
- **示例**: "按下返回键"

### 3. 应用管理工具

#### launch_app
- **功能**: 启动应用
- **参数**:
  - `package_name` (字符串): 应用包名
- **示例**: "启动微信应用"

### 4. UI元素操作工具

#### get_ui_elements
- **功能**: 获取界面上的UI元素
- **参数**:
  - `filters` (对象, 可选): 过滤条件
    - `text`: 按文字过滤
    - `class_name`: 按类名过滤
    - `resource_id`: 按资源ID过滤
- **示例**: "获取所有包含'登录'文字的按钮"

#### click_element
- **功能**: 点击UI元素
- **参数**:
  - `identifier` (字符串): 元素标识
  - `by` (字符串): 查找方式，可选值: "text", "class_name", "resource_id"
- **示例**: "点击'登录'按钮"

### 5. 性能监测工具

#### get_memory_info
- **功能**: 获取应用内存使用情况
- **参数**:
  - `package_name` (字符串): 应用包名
- **示例**: "查看微信的内存使用情况"

#### get_cpu_usage_by_package
- **功能**: 获取应用CPU使用率
- **参数**:
  - `package_name` (字符串): 应用包名
- **示例**: "查看微信的CPU使用率"

#### get_fps_info
- **功能**: 获取应用帧率信息
- **参数**:
  - `package_name` (字符串): 应用包名
- **示例**: "查看微信的帧率"

#### get_app_startup_time
- **功能**: 测量应用启动时间
- **参数**:
  - `package_name` (字符串): 应用包名
  - `count` (数字, 可选): 测试次数，默认3
- **示例**: "测试微信的启动时间"

#### get_logcat
- **功能**: 获取应用日志
- **参数**:
  - `package_name` (字符串): 应用包名
  - `lines` (数字, 可选): 日志行数，默认100
- **示例**: "查看微信的最近日志"

#### get_performance_snapshot
- **功能**: 获取应用性能快照
- **参数**:
  - `package_name` (字符串): 应用包名
- **示例**: "获取微信的完整性能数据"

#### monitor_performance
- **功能**: 持续监测应用性能
- **参数**:
  - `package_name` (字符串): 应用包名
  - `duration` (数字): 监测时长(秒)
- **示例**: "监测微信性能10分钟"

### 6. 广告分析工具

#### analyze_ad
- **功能**: 分析应用中的广告
- **参数**:
  - `package_name` (字符串): 应用包名
- **示例**: "分析微信中的广告信息"

## 🎯 使用技巧

### 自然语言提问示例
- "连接设备并启动微信" → 自动调用 `connect_devices()` + `launch_app('com.tencent.mm')`
- "点击登录按钮" → 自动调用 `click_element('登录', 'text')`
- "查看微信的内存使用" → 自动调用 `get_memory_info('com.tencent.mm')`

### 参数自动识别
- 系统会自动识别包名、按键名称等参数
- 不需要记住具体的参数格式

## 📱 常用应用包名
- 多盟: `com.domob1024.sdk.demo`
- bluex: `com.bluex.sdk.demo`

## 🔧 故障排除
- 如果工具调用失败，检查设备是否连接
- 确保应用包名正确
- 检查参数格式是否符合要求
